
# Healthcare Analysis Dashboard (Power BI)

## Project Overview
This project is a **Healthcare Analysis Dashboard** developed using **Microsoft Power BI**.
It provides insights into patient data, hospital visits, departments, and treatment costs to support data-driven decision-making in healthcare management.

## Contents
- `HEALTHCAE ANALYSIS DASHBORAD PROJECT.pbix` – Power BI dashboard file
- `Final Project.zip` – Supporting files / resources
- `README.md` – Project documentation

## Key Features
- Patient demographics analysis
- Department-wise visit analysis
- Treatment cost & revenue insights
- Interactive filters and slicers
- KPI cards for quick overview

## Tools & Technologies
- Microsoft Power BI
- Data Analytics & Visualization
- Healthcare Domain Dataset
